# Mi Repositorio de Manim 🎬

Aquí guardo el código fuente de mis animaciones. 
Para mantener el repositorio ligero, la carpeta `media/` con los videos renderizados se ignora automáticamente.

## Comandos útiles
- **Borrador rápido:** `manim -pql src/shorts/01_red_neuronal.py TuClaseScene`
- **Render final (1080p 60fps):** `manim -pqh src/shorts/01_red_neuronal.py TuClaseScene`
